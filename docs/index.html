アプリケーションについて

テキストファイルから特定の文字列を抽出し、新しいテキストファイルを生成するアプリケーションです。

実行環境

Java Runtime Environment 1.8

最新バージョン

Latest version: 1.0

ダウンロード

使い方

jtext-extraction.jarと同階層に、extract.listを作成する。 extract.listには、抽出したい文字列を改行区切りで記述する。※改行コードはJavaのSystem.lineSeparator()の戻り値としている。

改行コード

  • Windows: \r\n
  • macOS: \n
  • Linux: \n
